# TESTING_DATA Layout

Each codec folder uses the same structure:
- BitCrusher/<Category>/
- ShutterEncoder/<Category>/

Standard categories:
- Anime
- Cam_Recordings
- Film_Grain
- Gaming_Footage
- Low_Light
- Screen_Recordings
- Sports_HighMotion
- Static_Video
- Talking_Head


Suggested usage:
- Put each source clip in matching category names for both tools.
- Keep file naming parallel to simplify automated comparisons.
